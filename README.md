## 개요

- 랜선 여행은 집에서도 코로나와 같은 위험요소 없이
- 원하는 방법으로 원하는 장소로 원하는 것들을 보고,
- 느끼고 즐길 수 있는 여행을 제공함으로 실제 여행을
- 가지 못하는 고객들의 여행하고 싶어하는 요구를
- 충족 시켜주기 위한 사이트입니다.
